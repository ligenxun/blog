我想要猎杀一只龙

发表于 2015-09-18   |   分类于 龙物语   |  

“噗”

昏暗沉闷的小黑屋里回荡着一声重物落地的闷响，陈涛痛苦地看着眼前的一堆山的杂物。

“老妈，你确定爸他要的印章在这里面？”

“当然咯，就是这堆东西，嗯，你小时候不少的东西可是都在这里面呢！小时候的你可是比现在要可爱的多呢！”一声轻快的声音掠过的杂物堆的尘埃，落到陈涛的耳朵里，“你那个不靠谱的爸爸突然要做什么工艺品，说什么当年刻的一个章很有感觉，一定要拿来做灵感，没办法，只能靠你咯~”

一个风韵犹存的妇人小心地惦着脚，绕过地上的灰尘，笑眯眯地打量着自己的儿子。

“那老爸呢？”

“去台湾了，说什么要考虑考虑文化输出，要让岛上的人感受一下大陆文化的强盛。不过他貌似没考虑到其实台湾的艺术要比大陆发达，所以就死命地说当年刻的一个印章是自己才华的表现，一定要找出来。”妇人耸了耸肩，眼神轻轻掠过这堆杂物，无奈地说道，不过怎么都带着一股幸灾乐祸，“然后就要我来找咯，乖儿子，你不会要你美丽的妈妈去翻这么一堆东西吧~”

“你们这是坑儿！”陈涛无奈的摇了摇头，“天快黑了，找得到吗？老家这屋子现在可是住不下人的，要不明天？”

“没事啦，找得到明天不就不用回来了么！”妇人不在意地晃了晃手，指了指杂物，不言而喻。

“好吧好吧。”陈涛随意地蹲下，手指轻抚过满是灰尘的杂物们，似乎熟悉，又似乎陌生的感觉浮上心头。

“你真的不记得小时候的事情吗？”身后传来母亲的声音。

“都说是小时候的事情了，怎么会记得嘛！”陈涛压下心中的感觉，装作努力找东西的样子轻轻翻动着杂物。

“也就几年前的事情嘛，现在的小孩子，变的可真快。”母亲见儿子这么不给面子，一手捂住自己的鼻子，躲过翻动起来的灰尘，一边不满地嘟囔道。

“老妈你说什么？”陈涛随意地翻过一个纸板，上面不知道被谁画了一堆的符号，符号被细心地排列成圆形，圆形与圆形见嵌套形成一种奇异的立体感。陈涛随意地看了一眼，放到一边去，看见了一个黑色的纸箱子。有种心血来潮的，陌生又熟悉的感觉告诉自己，要找的东西就在下面。

“哼，我说小时候的你可比现在可爱多了。”

“是是是，小时候谁不可爱嘛！长大了这不都长残了么！真是对不起观众啊！”陈涛不在意地回复着，手指微微颤抖地划过纸箱的盖子。灰尘与划痕，微风送来了腐朽的味道，还有一个陌生的笔锋:

“我想要猎杀一只龙”

“啊！”

突然传来一身尖锐的尖叫，陈涛立刻跳了起来，回头看着自己的母亲尖叫地跑了出去，边跑还边叫喊:“老鼠啊！”

“哎哎哎，老妈，你就这样把你的儿子丢给老鼠了啊？”陈涛无奈地随手把小纸箱拿起，一边调侃自己的母亲，一边走出昏暗的杂物房，在灰尘的飘舞中走向一片光明。

外面的太阳已经开始缓缓地收敛起她的威严，周围是一片模糊的昏黄色，令人烦躁的蝉鸣也空灵了许多，破旧的老屋看着令人有种说不清道不明的味道。

“哼，等下你坐你伯父的摩托车下去！”在院子中平静下来的妇人很不满意自己刚刚的表现，踢着院子中肆虐的杂草，说道。

“不是吧！为什么我不能坐车？”陈涛很不满，看着明显打击报复的母亲，说道。

“我还要去走亲戚，你也去？”妇人瞥了一眼自己的儿子，说道。

“那我还是坐摩托车吧。。”陈涛想了想那些可怕的七大姑八大姨，很明智地放弃了坐车的想法。

路边传来了“滴滴”的车鸣。

“快快快，快去！”妇人立刻换了一副温婉的神态，一边一副赶人的模样，一边自己独自走向通往祖宅的小路。

“好吧，老妈你自己开车小心点！”陈涛摆了摆手，走出破旧的院子。抱着纸箱的手微微紧了紧，黑色的纸箱似乎会动地呼应着自己。

“‘我想要猎杀一只龙’么？”

真是中二病。

早上的梦

发表于 2015-09-18   |   分类于 Live   |  

早上醒来的时候，做了一个梦。梦见了离奇的故事。蛮有意思的，就写了下来

畸形兽，和几个人被拉入奇异空间，一切似乎都变得很诡异，一重梦境内大家都心想事成，得到自己渴望的东西，事业，美女，不久大家就为了维持梦境而在梦境中人的劝说下进入二层梦境，随后所有梦境人都变为畸形兽，疯狂对大家进行补杀，将死时刻突然一个和畸形兽很像的怪异巨汉出现，救下幸存者，并且带着大家沿着二层梦境光怪流离的梦境间穿梭，见到了沉默可怕的僵尸，红衣如火的女鬼，。。。到达了庇护所，庇护所内我们知晓了，原来巨汉是之前的幸存者，上一批幸存者死了只剩他一人了，而他也即将被同化成为畸形兽，为了留下火种，他冒险去和畸形兽抢人。然后和平了一段时间，期间有人为了去一层梦境而疯狂投入怪物中，有为了醒来自杀，有反正都是死不如享乐的，到巨汉召集去下一层的时候就剩寥寥几人，而三层要面对的是自己内心深处最不想要面对的事情，自己这辈子最愧疚的事情，只要你能够面对他们，那么三层就会成为你的梦境的叠加物，在剩下人中，随机挑出一人用他的梦境作为主体，其他人附加进入，第一次中年人，梦是平常的90年代初，他面对他的父亲，他农民老父亲要他回去结婚，他觉得自己已经长大了，有自己主见了，面对喋喋不休的父亲，他粗暴地拒绝了，要他父亲回家去，然后他父亲伤心地走了。回家的路上他父亲就死于车祸，他觉得是他害死了父亲，巨汉已经被畸形兽污染无法进入美梦部分，他刚刚经理了拒绝，大家都为了救下他父亲拼命去，付出了一个人被警察按到，一个人轻伤的代价，他们救下了他父亲，在大家庆幸的时候，他的父亲却突发心脏病死去了，他什么都没来得及说，他的梦境奔溃了，梦境人蜕变畸形兽，被摁倒的直接死亡，轻伤的在奔跑的时候摔倒，那个中年人，奔溃，被污染成僵尸，剩下两人。庇护所内巨汉再次强调，必须面对内心的愧疚，而不是试图去改变已经发生的。另一个人质问，但是为了回到现实，两人不得已再次进入下一层。他最愧疚的是他的初恋为了他去堕胎，应为黑诊所她再也不能怀孕了。茫然无助的她，看着另一个人，他把她拥入怀里，说没关系的，大不了以后领养一个，他一辈子和她不分开的。然后转头和我说，看，这是我老婆，现实中也是，所以，为了她，我必须回去啊。然后巨汉出现，梦境开始变化，黑诊所被放大，黑色的物质弥漫了地板，他的女友变成红衣女鬼，梦境中人变成了鬼，巨汉要大家不要在黑色物质中，期间他被拉入黑色物质中，红衣女鬼啃噬着他的胸膛，他怀抱着她，地上有小鬼一边“爸爸爸爸”地叫，一边啃噬他的小腿。他要我们走，他的女朋友已经应为产后抑郁和不能生育的打击自杀了，死亡是每一个人都要面对的，但他却给了她比死亡更痛苦的事情，堕胎，还有后面他对她的疏忽。这应该就是他的报应了。他被污染成为青衣怨灵。看了我一眼，笑了笑。(可沟通)

最后自己了，由于自己的倔强，和自己家人很不愉快，期间爷爷奶奶去世，痛苦地看着曾经的自己摔门而出，父母和老一辈担忧的目光，笑了笑，自己会和解的，无论逝者还是生者，他们都是我的亲人，最亲的人，然后进入三层，重重叠叠的屋顶，下面都是黑色物质，只要不是被黑色物质污染，就是我的梦境，不死的。生化畸形兽，畸形兽，追逐最后到一个大阳台的二层建筑，见到了三层的畸形主，冷漠地引爆了核弹。毒菇，貌似前几批幸存者之一，被污染后成为畸形助手。然后成千上万次死亡，死亡，似乎就是验证畸形的不可战胜。然后巴拉巴拉，由于巨汉更接近畸形兽，反复的梦境，巨汉快奔溃了，策反了小蘑菇，小蘑菇在畸形还没进入那栋楼以前发动了守护，还在经历愧疚考验的我迅速到哪里，乱七八糟战斗后，巨汉把自己美好记忆注入了畸形，畸形周围出现波动，我们乘机返回现实。巨汉却没回来。美好单纯的小蘑菇。醒来，打电话120去救乱吃蘑菇作死的小蘑菇一家人，然后打电话给父亲，今年回家吃年夜饭。想一个问题，大清早的，小蘑菇一家人吃蘑菇，也是蛮拼的。窗外朝阳正好。

离奇的梦，感觉像是另一个自己给自己寓言式的告诫。有趣。

秋雨

发表于 2015-09-14   |   分类于 Live   |  

下雨了。淅淅沥沥的，带着清新的气味，一点一点浇筑了我的哀愁。

这要怎么回去啊。

看着这雨没有停的意思，就拿了东西，下去先吃点去。吃着的时候看着外边的滴滴答答，想着要不要打车。直到吃完还没下决定。

于是最后还是骑车回去。

打着伞，尽量压低身子，把伞把和左手的车把一起握着，一下一下，小心翼翼地蹬着，还要小心地注意调皮的风不要掀起我的雨伞。车头还绑着今天送到的棉被，自行车后边的支柱还坏了，前进的时候还拖着凄惨的金属摩擦声。

过红绿灯的时候看着静静等待的车流，突然觉得自己好狼狈。

然后慢悠悠地蹬着车子从一群钢铁盒子前晃过。——管他呢！反正又没人认识我。

不成功的雨中西湖游记

发表于 2015-09-05   |   分类于 Live   |  

定风波
“莫听穿林打叶声，何妨吟啸且徐行。竹杖芒鞋轻胜马，谁怕？ 一蓑烟雨任平生。
料峭春风吹酒醒，微冷，山头斜照却相迎。回首向来萧瑟处，归去，也无风雨也无晴。”

今日未带伞，游西湖，被淋成落汤鸡。路过长堤，看着点点滴滴的湖面，遥望影影绰绰的雷峰塔，可惜我不是许仙，没有白素贞一起来雨中偶遇，更关键的是，没有伞，没有人帮忙撑伞。雨又大了，微风浮动，略冷。

雷峰塔居然长得是金碧辉煌的模样，好生失望，而且门票好贵，顿时没了兴致，不去了。

去苏堤路上，避雨路边小店，点了个十五的西瓜，三三两两，几口解决，然后看着人来人往，倒是别有风味。

雨中泛舟西湖，应该别有风味，而且貌似还有租可以自己划的船，更加有趣。不过自己一个人雨中划船，怎么看都是一副凄凉的风景，作罢。

“竹杖芒鞋轻胜马，谁怕，一蓑烟雨任平生”。突然间想起了苏轼，想起了同样是淋雨还傻乐地游玩的玩货，决定去看看苏轼纪念馆。

游苏轼纪念馆出来，天气放晴。

于是走到苏堤另一边。自己真是傻缺了，好远，然后疲惫地逛了逛一个植物园，就往回走。看见岳庙，朱门墙高，门票还贵，我这穷人消费不起。呵呵，现在还有景点什么是不收门票的？唉，希望收了门票能照顾好文物，不要和故宫那群人似地。

岳庙门口看了看，身子被风吹的有点冷，没了兴致，还是回去吧。

本想着今日能够畅游西湖的，天公不作美，淋了雨，心情更加不知从何道起。那就下次中秋再游一次西湖。

慢悠悠的骑行者

发表于 2015-09-04   |   分类于 Live   |  

慢悠悠地骑着自行车，嗯，杭州最美的地方就是这自由的借车系统了，虽然她的公交系统简直不能忍受，但是她的自行车方便啊。可以随意地骑行，累了就把车还了走路，走路累了就随意地找一个绿茵坐坐，回去要是不想骑车，那就坐公交回去，这可比自己买自行车要方便多，也自在多了。从这点上面看，我真是天才，哈哈，懒癌晚期又好玩。

有一脚没一脚地蹬着车子，慢悠悠地看着路边绿的很有层次的草木，路边的桂花书飘下点点的金黄，隐隐约约的花香，似有似无的蝉鸣，一切都如图试一样的美好。嗯，来杭州这点真的是超值了，体会了一把江南的早秋风景。

然后去了某个寺庙，名字又忘了。骑行了半天，上坡半天路，看着身后的不见头的下坡路，还是很有成就感的。然后累成狗地到了寺庙，就我一个推着租借的自行车，慢悠悠地晃过，到了寺庙门口，发现没地方停车，照两张照片，就滚蛋了。路上的风景已经看见了，至于这个终点，反正我不信佛，看看就好了，进不了就不强求了。
下山，老久，还车，过了时间了，乐极生悲。拿着张余额为零的市民卡，不，确切的说是欠费1元的市民卡，打了半天电话，然后再花看半天找到充值点，然后顺道办了张给公交卡，over，西湖是去不成了，那就明日吧。

啊，多么痛的领悟。。

算啦，没差啦。

然后就分开了，估计以后也没机会再次见面了。希望你能一路顺风，一直幸福，可爱的姑娘。
然后很累，一个人在9平米的被漆成白色的混凝土笼牢内，简直像是那个心理上面的酷刑，名字叫什么的忘了。
突然有点想恋爱了。
算了，一段时间内总会有这种奇怪的臆想，睡一觉就好了。
